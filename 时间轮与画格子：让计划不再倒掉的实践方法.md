> 整理日期：2026-03-29

## 写在前面：这是一篇讲什么的文章？

这篇文章解决一个核心问题：**为什么你的计划总是半途而废？**

作者给出了两个工具：
1. **画格子** —— 让你的努力看得见
2. **时间轮** —— 让你的计划容许中断

这两个工具可以单独使用，也可以配合使用。下面我分开讲清楚。

# 第一部分：画格子

## 一、画格子是什么？

**一句话定义**：
画格子就是把你的任务变成一个个小方格，贴在墙上，完成一个就划掉一个。

**核心作用**：
> **"要纠正无意识带来的认知偏差，你可以采用一些能够让自己的行为显化展示的手段，比如画格子。"**

简单说就是：**让努力看得见。**

---

## 二、具体怎么画格子？

### 步骤1：算出你需要多少个格子

**倒推法**：

```
问自己：我要完成什么大目标？
        ↓
问自己：给我多少时间？
        ↓
问自己：我总共需要做多少件事？
        ↓
问自己：平均每天要做多少件？
        ↓
得出：我需要画多少个格子
```

**具体例子**：

| 场景 | 目标 | 时间 | 总任务量 | 格子数 |
|-----|------|------|---------|-------|
| 备考 | 50篇练习 | 50天 | 50篇 | 50个格子 |
| 背单词 | 3000个单词 | 100天 | 每天30个 | 100个格子 |
| 读书 | 12本书 | 12个月 | 每月1本 | 12个格子 |

### 步骤2：准备格子

> **"用A4纸折出50个格子，把它贴在你抬头就可看的地方。"**

- 用A4纸折出对应数量的格子
- 贴在你抬头就能看到的地方（墙上、桌前）

### 步骤3：完成就划掉

> **"每写完一篇，就豪气划拉掉一个格，格子越大，划着就越有爽感。"**

- 完成一个任务，就划掉一个格子
- 格子画得越大，划起来越爽

### 步骤4：对上日期（可选但推荐）

> **"最好能够对上日期，这样这些格子就能够对你的任务进行跟进。"**

- 在格子里写上日期
- 可以看到自己的进度

## 三、三次打卡法：一件事情爽三次

**作者的做法**：

完成一件小任务，至少能让自己爽三次：

```
┌─────────────────────────────────────────────────────────┐
│  第一次爽：在墙上的格子里豪气划掉一个                      │
│  ↓                                                       │
│  第二次爽：在印象笔记里记录"今天完成了xx事"               │
│  ↓                                                       │
│  第三次爽：在微信签名里更新"目标完成动态"                  │
└─────────────────────────────────────────────────────────┘
```

**为什么可以爽三次？**

> **"我们所做的任何一件事情，它都不只是能够孤立地服务于单个目标。"**

**举例：背50个单词**

| 这个任务服务于 | 为什么算 |
|--------------|---------|
| 出国目标 | 背单词对出国有帮助 ✓ |
| 考试目标 | 可以考出更好成绩 ✓ |
| 交流目标 | 增加词汇量，更自信 ✓ |

**所以背一次单词，你可以觉得自己在三个方面都进步了——这就是自我效能感的来源。**

## 四、画格子的实践建议

1. **格子不要丢**：画完的格子聚拢张贴在墙上
2. **抬头可见**：贴在每天能看到的地方
3. **定期回顾**：偶一张望，过去做了多少事，了然于胸

> **"你偶一张望，过去做了多少实在的事情，立马了然于胸，这种感觉还是很舒服的。"**

# 第二部分：时间轮

## 五、时间轮是什么？

**一句话定义**：
时间轮就是一种"**用轮代替天**"的计划方式——不再规定每天必须做什么，而是规定"一轮"要完成什么，至于几天完成一轮，无所谓。

**核心变化**：

| 传统每日计划 | 时间轮计划 |
|-------------|-----------|
| 我每天必须读1小时书 | 我这一轮要读满8小时书 |
| 今天没做=失败了 | 今天没做？明天继续做就是了 |
| 按天计算进度 | 按"轮"计算进度 |

---

## 六、为什么要用时间轮？

### 问题：每日计划为什么会失败？

**作者的经验**：

> **"很多人平时给自己做行动计划，总会陷入「flag反复立反复倒」的怪圈。"**

**每日计划的致命问题**：

```
你给自己定了计划：每天必须读书1小时
        ↓
某天特别忙，到了晚上还没读
        ↓
开始焦虑："今天完不成了"
        ↓
要么敷衍了事读5分钟（只为了划勾）
要么直接放弃："算了，明天重新开始"
        ↓
计划失败
```

**"每日"的问题**：
- "每日"=一天为限，一天为宿命
- 一旦完不成，产生挫败感
- 中断一次就容易彻底放弃

### 时间轮如何解决？

**核心变化**：
> **"我不按「天」来算，我按照的是「轮」。"**

**新逻辑**：

```
你给自己定了计划：这一轮要读满8小时书
        ↓
第一天读了3小时（没满8小时）
        ↓
没关系，明天继续读
        ↓
第二天又读了5小时
        ↓
8小时满了！这一轮完成！
        ↓
开始下一轮
```

**容许中断**：
> **"真正牛逼的，不是连续3个月跑步不间断的人，而是那个因为出差一段时间没法跑，回来之后接着继续跑的人。"**

---

## 七、时间轮具体怎么设计？

### 作者的写作练习方案

**一轮包含什么？**

```
┌─────────────────────────────────────────┐
│           一轮 = 输入 + 输出              │
│                                            │
│  输入：累计阅读8小时                        │
│  输出：进行写作练习                          │
└─────────────────────────────────────────┘
```

**输出要求**：

| 情况 | 怎么办 |
|-----|-------|
| 能写出成型文章 | 尽情写，写多久都行（2-5小时都可以） |
| 写不出成型文章 | 硬性写满3小时，秒表计时，不赖一秒 |

**写不出怎么办？**

| 方法 | 具体做法 |
|-----|---------|
| 整理思维碎片 | 记录近来所思所想 |
| 以读助写 | 读别人文章，有想法了就开计时写 |

**一轮需要几天？**

> **"有时两天完成一轮，有时三五天完成一轮……至于最终需要多长时间，实则无定数——因时制宜，亦无须定数。"**

---

## 八、时间轮的三个核心特点

### 特点1：没有"每日"限制，消除焦虑

| 每日计划 | 时间轮 |
|---------|-------|
| 今天必须完成 | 这一轮完成就行，几天都无所谓 |
| 完不成就焦虑 | 今天没做完？明天继续 |
| 注意力在"今天" | 注意力在"总量" |

> **"新的计划理念，让我不会再把注意力集中在每日是否完成了既定的任务。"**

### 特点2：容许中断，继续就好

**为什么计划会因中断而失败？**

> **"我们设置的那些计划，很多时候之所以落得不了了之的下场，其实乃是因为「中断」所致。"**

**遇到中断时的情绪**：
- 打乱了节奏
- 产生挫败感、自责
- 算了，放弃吧

**时间轮的应对**：

> **"但凡遇到「中断」，最正确的做法是「继续就好」。"**

> **"当「中断」出现的时候，中断已经是一个客观现实，对于这样的现实，我们所能够采取的最好的应对方式就是接纳。"**

### 特点3：纯时间记录，只看累积

**什么是纯时间？**

- 只记录真正花在任务上的时间
- 上厕所、玩手机、发呆……这些时间都不算
- 用秒表计时，开始就计时，停下就暂停

**为什么用纯时间？**

> **"用纯时间来看待事物的观点，其实就是一种容许「中断」的观点。"**

**核心逻辑**：

```
我只关注：我的总时间有没有增加
        ↓
中断了？没关系，继续累积就是了
        ↓
"累积"何来"中断"之说？
```

**作者的比喻**：

> **"你可以想象我是一个贪婪的财奴，他守着财，勤劳的擦拭着自己的金币，并且他还总是想方设法的往金库里攒钱。"**

> **"我和守财奴唯一不同的地方在于：他的钱是可以被偷走的，但是我攒下的时间，就永永远远的属于自己，谁也偷不走。"**

## 九、输入输出为什么要绑定？

### 作者的配比

> **"每进行8小时的纯输入，就要进行3小时的纯输出。"**

### 为什么要绑定？

**作者的观点**：

> **"但凡不能够进行整合输出的输入，都是毫无意义的。"**

> **"一直不停的「输入」，其实是一种妄念。"**

**简单理解**：

| 只输入不输出 | 输入+输出绑定 |
|-------------|--------------|
| 看书→感觉学到了→其实没记住 | 看书→写文章→真正消化了 |
| 上网课→感觉收获满满→其实没掌握 | 上网课→做题→真正学会了 |
| 收藏干货→感觉不错→其实从来不用 | 收藏干货→实践→真正用上了 |

**坚持后的效果**：

> **"随着你严格输入量的增加，随着你严格的整合输出的训练的增加……你头脑中可以相互印证、相互碰撞的思想和案例也越来越多。"**

> **"意外的惊喜，可不就是「知识和知识的碰撞」所产生的么？"**

# 第三部分：时间记录的苛刻性

## 十、为什么时间记录必须严格？

### 作者的态度

> **"不管其背后的本质是什么，时间记录都需要苛刻的去做。"**

### 为什么不能掺水？

**如果掺水会怎样？**

- 你看数据时不会感到力量
- 你会对自己的记录失去信心
- 记录的"庄严性"被破坏

**星球成员的话**：

> **"但我坚持下来的几百个小时的时间轮，依然是我踏实努力的最好证明。"**

> **"如果她在这份记录上掺过水，则她说这句话的时候就会感到心虚。"**

---

## 十一、时间记录的三条原则

### 原则1：记录的项目越少越好

**作者的做法**：
> **"我只是集中记录跟自己写作相关活动的时间。"**

**为什么？**

| 啥都记录 | 只记录核心 |
|---------|-----------|
| 学习、写作、运动、吃饭都记 | 只记写作相关 |
| 压力大，都是任务 | 压力小，顺手记录 |
| 手忙脚乱 | 重点明确 |
| 相当于没有重点 | 集中做才能彻底 |

### 原则2：记录的"事"要够核心

**什么叫核心？**

1. **是你最大价值产生的地方**
   - 对博士：写论文
   - 对作者：写作

2. **死磕核心会带动其他**
   - 死磕写作 → 自然会阅读、思考、自律

> **"死磕核心目标，那么我就不得不去做其他的许多事情，这就是「一带多」。"**

### 原则3：一丝不苟，绝对严格

**作者的原则**：

> **"我会精确的记录到分钟。"**

> **"一般情况下，我就把那段时间给扔了。我绝不会把一个模糊的时间段记录到总时间里。"**

| 情况 | 处理方式 |
|-----|---------|
| 找不到起点时间 | 把那段时间扔了，不计入 |
| 模糊的时间段 | 绝不记录到总时间里 |
| 忘记计时 | 宁可放弃，绝不估计 |

**为什么必须这么严格？**

> **"精髓正在于计较，不计较就会放过自己。放过自己就会变得不在意。"**

> **"有第一次，就会有第二次。"**

> **"也许你做不到，首先可能并不是因为你的能力问题，而可能是因为第一步要做的事情你没做好，就是你对自己的要求进行妥协了。"**

# 第四部分：时间轮与画格子的关系

## 十二、两者有什么联系？

### 它们是两个独立的工具

| 维度 | 画格子 | 时间轮 |
|-----|-------|-------|
| **本质** | 可视化工具 | 计划理念 |
| **解决什么问题** | 让努力看得见 | 让计划容许中断 |
| **关注点** | 任务完成数量 | 时间累积总量 |
| **时间单位** | 天（每天划一个） | 轮（完成一轮闭环） |
| **核心价值** | 三次打卡的爽感 | 中断了也不算失败 |

### 但它们可以配合使用

**配合方式一**：用画格子记录完成的"轮"

```
比如：你要完成10轮"输入+输出"
        ↓
画10个格子，贴墙上
        ↓
完成一轮，划掉一个格子
        ↓
10个格子都划掉了 → 目标达成！
```

**配合方式二**：用时间轮指导画格子

```
比如：你要用时间轮的方式学英语
        ↓
算出：总共需要完成30轮（每轮=输入5小时+输出2小时）
        ↓
画30个格子，贴墙上
        ↓
完成一轮，划掉一个格子
```

**总结**：
- **画格子**是"展示板"——让你看到进度
- **时间轮**是"规则书"——告诉你怎么计划

# 第五部分：具体怎么开始实践

## 十三、从0开始的实践步骤

### 方案A：只想用画格子

**适合：目标清晰，任务固定的场景（如背单词、做习题）**

```
步骤1：确定目标
   比如：50天背完3000个单词

步骤2：倒推任务
   每天60个 → 50天 = 50个格子

步骤3：画格子
   A4纸折50个格子，贴墙上

步骤4：开始执行
   每天背完，划掉一个格子

步骤5：三次打卡（可选）
   格子划掉 + 笔记记录 + 签名更新
```

### 方案B：只想用时间轮

**适合：任务灵活、容易中断的场景（如阅读、写作、练技能）**

```
步骤1：确定一轮包含什么
   比如：阅读8小时 + 写作3小时 = 一轮

步骤2：开始第一轮
   用秒表计时，开始阅读

步骤3：容许中断
   今天读了3小时？没关系，明天继续

步骤4：完成一轮
   8小时读完，开始写作3小时
   写完 → 一轮完成！

步骤5：开始下一轮
   循环往复
```

### 方案C：两者配合使用

**适合：长期项目，需要持续激励**

```
步骤1：确定总目标
   比如：3个月完成10轮"输入+输出"

步骤2：画格子
   画10个格子，代表10轮

步骤3：按时间轮执行
   一轮一轮地完成

步骤4：划格子激励
   每完成一轮，划掉一个格子

步骤5：回顾进度
   看墙上的格子，知道自己走到哪了
```

## 十四、常见问题

### Q1：我是上班族/学生，适合用哪个？

| 场景 | 推荐 |
|-----|------|
| 备考（刷题、背书） | 画格子 |
| 学习技能（阅读、写作） | 时间轮 |
| 长期项目（论文、作品） | 两者配合 |

### Q2：一轮应该设置多长时间？

**作者的建议**：
- 输入8小时 + 输出3小时（作者的比例）
- 你可以自己调整，比如4:1、3:2

**原则**：
- 不要只输入不输出
- 输出必须有硬性要求（时间或数量）

### Q3：忘记记录时间了怎么办？

**作者的做法**：
> **"一般情况下，我就把那段时间给扔了。我绝不会把一个模糊的时间段记录到总时间里。"**

**核心**：宁可少记，绝不估计。

### Q4：一天能累计多少纯时间？

**作者的实际情况**：

| 纯时间 | 感受 |
|-------|------|
| 6小时 | 已经满意 |
| 7-8小时 | 今天可真拼 |
| 9小时 | 好像一整天都在干这事 |
| 10-12小时 | 极端情况，很少见 |

> **"所以有人说自己干了一整天的工作，或者说自己经常一天干十多个小时，我是不太敢相信的。"**

### Q5：中断了很久怎么办？

**答案**：继续就好。

> **"真正牛逼的，不是连续3个月跑步不间断的人，而是那个因为出差一段时间没法跑，回来之后接着继续跑的人。"**
