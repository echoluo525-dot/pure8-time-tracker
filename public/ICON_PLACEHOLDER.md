# 图标说明

## 需要添加的图标文件

请在此目录下添加以下图标文件：

### 1. icon-192.png
- 尺寸：192x192 像素
- 格式：PNG
- 用途：Android 主屏幕图标

### 2. icon-512.png
- 尺寸：512x512 像素
- 格式：PNG
- 用途：高分辨率设备图标

## 图标设计建议

- 使用简洁的设计
- 主色调：#1890ff（蓝色）
- 可以使用 ⏱ 或 ▦ 作为图标元素
- 确保在小尺寸下也能清晰识别

## 生成工具

可使用以下工具生成图标：
- Figma
- Canva
- 在线图标生成器

## 临时方案

开发阶段可以使用纯色方块或文字作为临时图标。
